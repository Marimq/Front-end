# Booble Kiss

Booble Kiss é um site front-end criado para representar uma marca fictícia de moda e lifestyle.  
O projeto foi desenvolvido com foco em layout visual, organização de conteúdo e adaptação para diferentes tamanhos de tela.

O site apresenta uma página inicial com imagens, seções de destaque e elementos visuais pensados para uma experiência simples e moderna.

## Tecnologias utilizadas

- HTML5  
- CSS3  

Todo o projeto foi feito sem frameworks, utilizando apenas HTML e CSS puro, com atenção à estrutura do código e à responsividade.

## Objetivo do projeto

Este projeto foi criado para praticar desenvolvimento front-end, organização de arquivos e construção de layouts responsivos, servindo também como parte do portfólio pessoal.
